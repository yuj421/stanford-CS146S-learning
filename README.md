# 🚀 CS146S: The Modern Software Developer 学习历程

欢迎来到我的 **CS146S (现代软件开发人员)** 学习项目！

本仓库记录了我学习斯坦福大学 2025 年秋季课程 **[CS146S: The Modern Software Developer](https://themodernsoftware.dev/)** 过程。在这里，整理了官方的阅读资料和讲义 PPT，还会分享学习笔记、实战作业以及踩坑经验。

## 🧠 课程核心理念

过去几年，大语言模型（LLMs）为软件开发引入了革命性的新范式。这门课的核心不是教我们如何“偷懒”或盲目依赖 AI（课程明确反对无脑的 "Vibe Coding"），而是教我们如何：
- 将生产力提升 10 倍（10x productivity）。
- 从传统的“代码劳工”转型为 **“AI Agent 管理者” (AI Agents Manager)**，坚持“人机协同工程 (Human-Agent Engineering)”。
- 掌握从计划、AI 生成、修改到迭代的全新现代开发工作流。

## 📂 仓库目录结构

- `readings/`: 官方推荐的深度阅读材料（PDF 或 Markdown 整理版）。

## 🛠️ 环境配置指南 (Repo Setup)

官方课程项目基于 Python 3.12 和 Poetry 进行依赖管理。请按照以下步骤配置本地开发环境：

1. **安装 Anaconda**：下载并安装 Anaconda Individual Edition，确保 `conda` 命令可用。
2. **创建并激活 Conda 环境**：建议创建一个 Python 3.12 的虚拟环境。
3. **安装 Poetry**：用于现代 Python 项目的依赖管理。
4. **安装项目依赖**：在激活的 Conda 环境下，进入仓库根目录，运行 Poetry 安装依赖。

## 📅 10周学习打卡地图

- [ ] **Week 1: 编码 LLM 与 AI 开发入门** (Introduction to Coding LLMs and AI Development)
  - 核心：LLM 原理与高效提示词工程 (Prompt Engineering)。
- [ ] **Week 2: 编码 Agent 解剖学** (The Anatomy of Coding Agents)
  - 核心：Agent 架构、工具调用与模型上下文协议 (MCP)。
- [ ] **Week 3: AI IDE 与上下文工程** (The AI IDE)
  - 核心：上下文管理、为 Agent 编写 PRD (Specs Are the New Source Code)。
- [ ] **Week 4: 编码 Agent 模式** (Coding Agent Patterns)
  - 核心：管理 Agent 的自治级别与人机协作模式 (如使用 Claude Code)。
- [ ] **Week 5: 现代终端** (The Modern Terminal)
  - 核心：AI 增强的命令行交互与自动化 (如 Warp 终端)。
- [ ] **Week 6: AI 测试与安全** (AI Testing and Security)
  - 核心：安全的 AI 编程、漏洞检测与防御 Agent 越权。
- [ ] **Week 7: 现代软件支持** (Modern Software Support)
  - 核心：AI 代码审查 (Code Review)、调试诊断与智能文档生成。
- [ ] **Week 8: 自动化 UI 与应用构建** (Automated UI and App Building)
  - 核心：全民设计与前端开发、快速 UI/UX 原型迭代。
- [ ] **Week 9: 部署后的 Agent 运维** (Agents Post-Deployment)
  - 核心：AI 系统的监控、可观测性与自动化事件响应。
- [ ] **Week 10: AI 软件工程的未来** (What's Next for AI Software Engineering)
  - 核心：新兴 AI 编码范式与行业趋势预测。
- [ ] **Final Project: 期末大项目**
  - 占官方评分 80% 的实战项目，展示现代开发实践。
