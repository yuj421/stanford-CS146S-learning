# 第 1 周：编码 LLM 与 AI 开发入门 (Introduction to Coding LLMs and AI Development)

## 🎯 核心目标 (Core Objective)
本周的核心在于**认知的重构**：要求我们将视角从单纯的“AI 用户”转变为严谨的“工程师”。我们要抛弃随意的习惯，将提示词工程 (Prompt Engineering) 视为一门严谨的科学方法论。通过本周学习，你需要建立起工程化的实验思维——即“提出假设 -> 设计 prompt -> 观察结果 -> 迭代优化”。

## 📝 详细主题 (Topics)
*   **课程安排与逻辑 (Course logistics)**：了解课程的整体目标，即现代软件开发者如何利用 AI 工具将生产力提升 10 倍。
*   **LLM 到底是什么 (What is an LLM actually)**：深入理解编码大语言模型 (Coding LLMs) 的底层机制，了解它是如何理解上下文和生成代码的。学会在面对复杂算法需求时，预判模型的潜在盲点，从而设计防御性指令。
*   **如何有效进行 Prompt (How to prompt effectively)**：将提示词工程从“凭感觉”转变为可量化、可复现的科学实验。掌握一套评估模型编码能力的体系，科学度量 AI 的输出质量。

*   **LLM 提示词实践平台 (LLM Prompting Playground)**
